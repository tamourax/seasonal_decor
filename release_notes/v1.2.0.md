# seasonal_decor v1.2.0

## Highlights

- Add emotional seasonal greeting text overlay with animation controls:
  - `showText`, `text`, `textStyle`, `textOpacity`
  - `textAlignment`, `textPadding`
  - `textDisplayDuration`, `textAnimationDuration`, `textSlideOffset`
- Add backdrop layering controls:
  - `showBackgroundBackdrops`
  - `showDecorativeBackdrops`
  - `backgroundBackdrop` (custom widget replacement for built-in background layer)
- Add runtime visual tuning:
  - `particleSizeMultiplier`
  - `decorativeBackdropDensityMultiplier`
- Improve fireworks replay behavior so `repeatEvery` restarts feel immediate
  for presets like Eid al-Adha.
- Upgrade demo screens (`main.dart` + `advanced_main.dart`) with richer live
  controls for atmosphere and greeting text.
