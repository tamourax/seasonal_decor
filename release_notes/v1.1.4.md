# seasonal_decor v1.1.4

## Highlights

- Added two new intensity levels: `DecorIntensity.extraHigh` and `DecorIntensity.max`.
- Added runtime speed control with `particleSpeedMultiplier`.
- Added theme-aware rendering via `adaptColorsToTheme`.
- Improved light-mode backdrop visibility for subtle presets.
- Added per-shape speed overrides via `SeasonalPreset.withOverrides(shapeSpeedMultipliers: ...)`.
- Added widget-level preset override params in `SeasonalDecor` for full runtime control.
