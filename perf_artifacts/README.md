# Performance Artifacts

Store exported timeline captures and session reports here.

Expected structure:

1. `perf_artifacts/<yyyy-mm-dd>/android/S0_run1.json`
2. `perf_artifacts/<yyyy-mm-dd>/web/S0_run1.json`
3. `perf_artifacts/<yyyy-mm-dd>/<platform>_baseline_report.md`

Raw artifacts are ignored by git. Keep permanent summaries in `docs/performance/`.
